# ğŸ“ˆ Formula Used
The calculator uses the standard compound interest formula:

      A = P(1+r/n)^(nt)
 
Where:

A = final amount

P = principal amount

r = annual interest rate (decimal)

n = number of compounding periods per year

t = number of years
